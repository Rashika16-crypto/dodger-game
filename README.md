# Dodger Game

A small browser game built with HTML and JavaScript.

## How to play
- Open `index.html` in your browser.
- Use the left and right arrow keys to move the player.
- Avoid falling blocks.
- Click **Restart** to play again.

## Files
- `index.html` — game page and styling
- `new.js` — game logic and controls

## Publish on GitHub
1. Create a new repository on GitHub.
2. Add this project folder to the repo and push it.
3. In the repository settings, enable **GitHub Pages**.
4. Set the source branch to `main` (or `master`) and root folder `/`.
5. After a few moments, GitHub Pages will provide a URL to open the game.

## Notes
- No build step is required.
- The game runs entirely in the browser.
